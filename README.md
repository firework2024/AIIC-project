# AIIC Project

本仓库当前主项目是 `mock-interviewer`：一个面向金融行业求职场景的智能模拟面试助手。

线上演示地址：

```text
https://8.140.221.115/
```

## 主项目

[mock-interviewer](mock-interviewer/) 提供金融求职模拟面试能力，包括：

- JD 定制化出题
- 简历深挖
- 业务模拟
- 混合面试
- 作品答辩
- 限时面试
- 压力面指数
- 候选人明确问题
- 逐题评分与最终报告
- 本机历史记录与知识卡片库

详细功能介绍、技术架构和本地运行方式请查看：

[mock-interviewer/README.md](mock-interviewer/README.md)

## 其他目录
