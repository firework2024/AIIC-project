## Chatbot Deployment Demo

The current public deployment demo lives in [`deploy-demo/`](deploy-demo/).

It provides:

- A FastAPI backend proxy for DeepSeek chat completions
- A simple browser chatbot UI
- Nginx and systemd deployment templates
- A `/health` endpoint and `/deploy-info` deployment timestamp endpoint
